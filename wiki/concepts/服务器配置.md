# 服务器配置

服务器配置是运维工作的核心，涵盖从初始化到安全加固的全流程。

## 核心配置项

### 远程访问

- [[SSH 免密登录]] — 安全的无密码远程登录方式
- SSH 端口配置与安全策略
- 防火墙规则设置（iptables/ufw）

### 用户与安全

- 用户与权限管理
- sudo 配置
- SELinux/AppArmor 配置
- 定期安全更新

### 网络与服务

- 静态 IP 配置
- DNS 配置
- 时区与 NTP 同步
- 服务管理（systemd）

## 常用工具

| 工具 | 用途 |
|------|------|
| ssh | 远程登录 |
| scp/sftp | 文件传输 |
| rsync | 增量同步 |
| htop/top | 性能监控 |
| netstat/ss | 网络诊断 |
