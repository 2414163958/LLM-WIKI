# Obsidian CLI

Obsidian 官方命令行工具，用于与 Vault 交互、插件和主题开发。

## 功能

- Vault 操作
- 插件开发
- 主题开发
- 批量操作

## 文档

[Obsidian CLI 官方文档](https://help.obsidian.md/cli)

## 在 Agent Skills 中的应用

[[obsidian-skills]] 提供的 `obsidian-cli` skill 让 Agent 能够：
- 通过命令行操作 Vault
- 开发和测试插件
- 管理主题
- 执行批量操作
