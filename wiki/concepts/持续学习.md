# 持续学习

从 AI 会话中自动提取模式并转化为可复用技能的系统化方法。

## 核心思想

AI 每次会话都会产生有价值的行为模式，但传统上这些模式会随会话结束而丢失。持续学习系统将这些模式**捕获、评估、聚类、固化**，形成可复用的知识资产。

## Everything Claude Code 的实现

### Instincts → Skills 转化流程

| 阶段 | 内容 | 说明 |
|------|------|------|
| **Instinct** | 单次会话学到的直觉 | 带置信度评分（0-1） |
| **Evaluate** | 评估直觉的适用性和质量 | `/learn-eval` |
| **Evolve** | 将相关直觉聚类为技能 | `/evolve` |
| **Skill** | 固化为可复用的工作流 | SKILL.md 文件 |

### 命令体系

| 命令 | 功能 |
|------|------|
| `/instinct-status` | 显示已学习直觉及其置信度 |
| `/instinct-import` | 从他人导入直觉 |
| `/instinct-export` | 导出直觉以供分享 |
| `/evolve` | 将相关直觉聚类为技能 |
| `/promote` | 将项目级直觉提升为全局直觉 |
| `/prune` | 删除过期待处理直觉 |

## v2 版本创新

基于**本能（Instinct）**的学习，而非显式规则：

- 每个直觉带置信度评分
- 低置信度的直觉可被淘汰
- 高置信度的直觉自动聚类为技能
- 支持团队间分享直觉集

## 与其他方法的对比

| 方法 | 知识来源 | 持久化方式 |
|------|----------|------------|
| 传统知识管理 | 手动编写 | 静态文档 |
| RAG | 原始文档 | 每次重新检索 |
| LLM Wiki | 原始文档 → Wiki | 增量构建 Wiki |
| 持续学习 | 会话行为 | Instincts → Skills |
