# TUI

Terminal User Interface（终端用户界面），基于文本的交互式界面。

## 定义

TUI 是在终端中运行的图形化用户界面，使用文本字符和颜色渲染，而非图形窗口系统。

## 特点

- **轻量级**：无需 GUI 环境，资源占用低
- **远程友好**：通过 SSH 即可使用
- **键盘驱动**：高效键盘操作，无需鼠标
- **可组合**：与其他 CLI 工具管道集成

## 在 AI 编程工具中的应用

[[OpenCode]] 由 Neovim 爱好者和 terminal.shop 创建者打造，聚焦 TUI 界面：

- 追求终端体验的极致
- 可本地运行，远程控制
- TUI 只是众多潜在客户端之一（客户端/服务器架构）

## 相关工具

- Neovim — TUI 文本编辑器
- tmux — TUI 终端复用器
- lazygit — TUI Git 客户端
