# BRAT

Beta Reviewer's Auto-update Tool，Obsidian 社区插件，用于安装和自动更新测试版插件。

## 用途

- 安装尚未发布到社区市场的插件
- 自动检测并更新 Beta 版本插件
- 支持从 GitHub 仓库直接安装

## 使用方法

1. 从 Obsidian 社区插件安装 BRAT
2. 打开 BRAT 设置，点击"Add Beta plugin"
3. 输入 GitHub 仓库路径：`用户名/仓库名`
4. 点击"Add Plugin"

## 典型应用

- [[opencode-obsidian]] 插件通过 BRAT 安装
- 测试开发中的插件版本
