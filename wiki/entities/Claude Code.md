# Claude Code

[[Anthropic]] 开发的 AI 编程 CLI 工具，是 [[oh-my-openagent]] 的启发来源之一。

## 基本信息

- **开发者**: Anthropic
- **定位**: AI 编程 CLI 工具
- **官网**: https://code.claude.com/docs/overview

## 核心特性

- Hooks 系统
- 命令系统
- 技能系统（Skills）
- MCP 服务器集成
- 插件系统

## 与 oh-my-openagent 的关系

- oh-my-openagent 受 Claude Code 深度启发
- oh-my-openagent 把 Claude Code 好用的特性搬过来并底层强化
- oh-my-openagent 完全兼容 Claude Code 的 Hooks、命令、技能、MCP、插件

## 与 OpenCode 的对比

详见 [[AI 编程工具对比]]

## ECC 生态

[[Everything Claude Code]] (ECC) 是 Claude Code 的完整配置集合：
- 140K+ stars，Anthropic 黑客马拉松获胜者
- 47 个代理、181 个技能、79 个命令
- 持续学习系统、内存持久化、AgentShield 安全审计
- 跨平台支持 OpenCode、Cursor、Gemini 等
