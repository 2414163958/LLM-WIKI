# Upstash

无服务器云服务提供商，提供 Redis、Kafka、QStash 等托管服务，开发了 [[Context7]]。

## 产品线

| 产品 | 类型 | 说明 |
|------|------|------|
| Redis | 数据库 | 无服务器 Redis |
| Kafka | 消息队列 | 无服务器 Kafka |
| QStash | 消息队列 | 无服务器消息队列 |
| Context7 | AI 工具 | LLM 文档检索平台 |

## Context7 项目

[[Context7]] 是 Upstash 开发的开源项目（MIT 协议），为 LLM 和 AI 代码编辑器提供最新的代码文档。

## 特点

- **Serverless 优先**：按需付费，无需管理基础设施
- **边缘计算友好**：低延迟全球分布
- **开发者体验**：简单易用的 API 和 SDK
