# JSON Canvas

开放的无限画布文件格式（`.canvas`），用于创建节点、边、分组和连接的可视化结构。

## 特点

- 开放标准，定义于 [jsoncanvas.org](https://jsoncanvas.org/)
- 支持节点、边、分组、连接
- 被 Obsidian 原生支持
- 可被 Agent 通过 `json-canvas` skill 创建和编辑

## 在 Agent Skills 中的应用

[[obsidian-skills]] 提供的 `json-canvas` skill 让 Agent 能够：
- 创建 `.canvas` 文件
- 操作节点、边、分组
- 构建可视化知识图谱
